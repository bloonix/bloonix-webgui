Benutzerdefinierte Charts

Die Plugins von Bloonix bieten bereits eine Menge vordefinierter Charts zur Ansicht von Statistiken. Diese Charts sind für die Sicht auf einen einzelnen Service sehr hilfreicht, es kann jedoch auch vorkommen, dass man gerne eigene Charts erstellen möchte, zum Beispiel um in einem Chart die CPU-Auslastung mehrerer Server gleichzeitig zu sehen. Für diesen Zweck gibt die Möglichkeit der Benutzer-Charts.

Um einen eigenen Chart zu erstellen, navigieren Sie zunächst in den Bereich User-Chart-Editor:

%%site.help.doc.bloonix-agent-installation%%

Grundlegendes

Zur Installation des Bloonix-Agenten auf Ihrem Server haben Sie zwei Möglichkeiten:

  1. Installation aus dem Repository
  2. Installation aus den Sourcen

Wenn Sie die Distributionen Debian, Ubuntu, Red Hat oder CentOS einsetzen - oder auch ähnliche Distributionen -, dann können Sie den Bloonix-Agenten und die Plugins aus einem Repository installieren und auch über den Paketmanager aktualisieren.

Installation auf Debian/Ubuntu

Da das Bloonix-Repository über HTTPS erreichbar ist, sollten Sie vorab das folgende Paket installieren: 

apt-get install apt-transport-https

Desweiteren müssen Sie den GPG-Key von Bloonix und das Repository einbinden: 

wget https://download.bloonix.de/debian/bloonix.gpg -O- | apt-key add -

Tragen Sie nun folgende Zeile in der Datei /etc/apt/sources.list ein: 

echo "deb https://download.bloonix.de/debian/ wheezy main" >>/etc/apt/sources.list

Sie können das Wheezy Repository für alle Debian und Ubuntu Versionen einsetzen.

Nun können Sie nach Paketen von Bloonix suchen:

apt-get update
apt-cache search bloonix

Installieren Sie nun den Bloonix-Agenten und die Plugins, die Sie benötigen. Zum Beispiel 

apt-get install bloonix-agent bloonix-plugins-linux bloonix-plugins-mysql

Installation auf CentOS/Red Hat

Die Installation auf CentOS oder Red Hat Systemen ist sehr einfach. Binden Sie zunächst das Repository von Bloonix ein: 

rpm -ivh \
    https://download.bloonix.de/centos/6/noarch/bloonix-release-0.3-1.el6.noarch.rpm

Daraufhin können Sie sofort nach Bloonix Paketen suchen: 

yum search bloonix

Installieren Sie nun den Bloonix-Agenten und die Plugins, die Sie benötigen. Zum Beispiel 

yum install bloonix-agent bloonix-plugins-linux bloonix-plugins-mysql

Installation des Agenten aus den Sourcen

Auch die Installation aus den Sourcen ist sehr einfach, wenn die Bloonix Pakete für Ihre Distribution nicht zur Verfügung stehen. Der Pfad zu den Sourcen lautet: 

https://download.bloonix.de/sources/

Die Installation läuft wie folgt ab: 

wget https://download.bloonix.de/sources/bloonix-agent-0.21.tar.gz
tar -xzf bloonix-agent-0.21.tar.gz
cd bloonix-agent-0.21
make
make install

Bitte beachten Sie das die Version des Agenten eventuell schon weiter fortgeschritten ist.

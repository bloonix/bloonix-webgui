Grundlegendes

Zur Einrichtung eines neuen Hosts sind mehrere Schritte notwendig, die im Einzelnen beschrieben werden.

Zunächst erhalten Sie eine kleine Übersicht, welche Schritte Sie durchführen müssen:

  1. Im Bereich MONITORING ⇒ Hosts muss der Host angelegt werden.
  2. Der Host muss einer Gruppe und sollte auch einer Kontaktgruppe hinzugefügt werden.
  3. Es müssen Services für den Host angelegt werden.
  4. Die Installation des Bloonix-Agenten ist notwendig, falls Sie die Systemvitals Ihres Hosts überwachen möchten.

Die Einrichtung im Detail

Zu 1.

Den neuen Host können Sie im Bereich MONITORING ⇒ Hosts anlegen. Oben Rechts befindet sich ein Plus-Icon, um zum Formular zu gelangen, mit dem Sie einen neuen Host anlegen können.

Dort angelangt, füllen Sie das Formular mit allen notwendigen Daten aus. Am wichtigsten hierbei sind der Hostname, die IP-Adresse und eine kleine Beschreibung des Hosts. Das Passwort, welches die Bloonix-Agenten zur Authentifizierung verwenden müssen, sollte bereits mit einem zufälligen Passwort generiert worden sein. Sind alle Felder ausgefüllt, so können Sie den Host anlegen.

Zu 2.

Im gleichen Formular zur Anlegung eines Hosts können Sie auch gleichzeitig eine Gruppe und eine Kontaktgruppe auswählen, in die der Host nach dem Anlegen hinzugefügt werden soll. Es ist wichtig, dass der Host einer Gruppe zugeordnet wird, da er ansonsten für die Benutzer nicht im Bereich MONITORING sichtbar ist. Über die Kontaktgruppen wird geregelt, welche Kontakte eine Benachrichtigung per E-Mail oder SMS bei kritischen Ereignissen eines Hosts erhalten.

Zu 3.

Alles, was dem Host jetzt noch fehlt, sind Services, die überwacht werden sollen. Klicken Sie hierzu in den Bereich MONITORING und wählen Sie den eben angelegten Host aus. In der linken Navigationsleiste öffnet sich ein Menü speziell für diesen Host. Klicken Sie dort auf Services und wählen Sie aus, ob Sie einen neuen Service aus einem vordefiniertem Template einrichten möchten oder einfach nur einen neuen Service ohne Template.

Weitere Informationen zur Einrichtung von Services finden Sie im Dokument: %%site.help.doc.add-new-service%%

Zu 4.

Ist der Host mit den Services nun in der WebGUI konfiguriert, so können Sie den Bloonix-Agenten auf Ihrem Server installieren, vorausgesetzt natürlich, Sie möchten die Systemvitals Ihres Servers überwachen. Eine Anleitung zur Installation des Agenten finden Sie hier: %%site.help.doc.bloonix-agent-installation%%

Bitte beachten Sie, dass Sie zur Einrichtung von Hosts und Services und des Bloonix-Agenten jederzeit den Support von Bloonix kostenfrei beauftragen können, wenn Sie als Produkt den Cloud-Service von Bloonix in Anspruch nehmen.
